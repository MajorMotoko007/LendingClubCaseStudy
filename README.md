# Lending Club Case Study 


## EDA On Lending Club Loan Data Set 
### Problem Statement 


``` 
Lending loans to ‘risky’ applicants is the largest source of financial loss
(called credit loss). The credit loss is the amount of money lost by the lender 
when the borrower refusesto pay or runs away with the money owed. ``` 
## Analysis Approach  
``` To Identify consumer attributes and loan attributes influencing towards the tendency of the default of the applicant, identifying these risky loan applicants will help such reduces such risky loans thereby cutting down the amount of credit loss. 
Identification of such applicants using EDA is the aim of this case study.   

Perform an analysis to understand the driving factors (or driver variables)
behind loan default, i.e.the variables which are strong indicators of default.  
The company can utilise this knowledge for its portfolio and risk assessment. 

```
